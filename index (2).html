Key Findings & Recommendations
01
SMB Signing Disabled — 10.0.0.100
Server Message Block (SMB) message signing is not enforced. An attacker on the same network segment could perform NTLM relay attacks, potentially gaining unauthorized access to shared resources without valid credentials.
MITRE ATT&CK: T1557.001 — LLMNR/NBT-NS Poisoning and SMB Relay
HIGH
02
RDP Exposed on Network — 10.0.0.100
Remote Desktop Protocol (port 3389) is open and reachable. RDP is a common target for brute-force and exploitation attacks. If not required, this service should be disabled or restricted to specific IPs via firewall rules.
MITRE ATT&CK: T1021.001 — Remote Desktop Protocol
HIGH
03
Unencrypted HTTP on Router — 10.0.0.1
The router admin interface is accessible over plain HTTP (port 80) in addition to HTTPS. Credentials transmitted over HTTP could be intercepted via man-in-the-middle attacks. Recommend disabling HTTP and enforcing HTTPS only.
MITRE ATT&CK: T1557 — Adversary-in-the-Middle
